## -> User APIs
 route | method | description |
| ----------- | ----------- | ----------- | 
| `/api/users/get/all` | ***GET*** | get all users
| `/api/users/:id` | ***GET***  | get user by id
| `/api/users/create` | ***POST*** | create new user
| `/api/users/:id` | ***PUT***  | update user details
| `/api/users/:id` | ***DELETE***  | delete a user
